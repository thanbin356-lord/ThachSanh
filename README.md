# Thach Sanh - 3D Action Adventure Game

Thach Sanh is a 3D action-adventure game inspired by the Vietnamese folk tale “Thạch Sanh”.
The project was developed using Unreal Engine 5 with Blueprint systems.

In this project, I was responsible for:

* Main gameplay programming using Blueprint
* Environment creation
* Level design
* Quest design and progression
* Character and player stats balancing
* Lighting, post-processing, and scene atmosphere
* Gameplay setup and polishing
* Creating the official trailer video

The game focuses on action combat, exploration, and bringing a traditional Vietnamese story into a modern 3D game experience.

Through this project, I improved my skills in:

* Unreal Engine 5 workflow
* Blueprint scripting
* Environment art and optimization
* Quest and gameplay design
* Cinematic presentation and trailer editing

## Technologies Used

* Unreal Engine 5
* Blueprint System
* Blender
* Premiere Pro / After Effects

## Project Purpose

This project was created as a graduation project and as an opportunity to gain more experience in building a complete 3D action-adventure game pipeline.
